# Tugas React - Halaman Katalog Produk

Repository tugas React untuk membuat halaman katalog produk.

## Fitur yang Berhasil Dikerjakan

✅ Menampilkan daftar produk

✅ Pencarian produk

✅ Filter kategori

✅ Responsive layout

✅ React Functional Component

✅ React Hooks

## Cara Menjalankan

```bash
npm install
npm run dev
```

## Teknologi

- React
- Vite
- JavaScript
- CSS
